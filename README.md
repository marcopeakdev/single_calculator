<!-- Getting Started -->
## 	Getting Started

<!-- Run Locally -->
### Run Locally

Clone the project

```bash
  git clone https://github.com/natainditama/calcway.git
```

Go to the project directory

```bash
  cd calcway
```

Install dependencies

```bash
  npm install
```

Start the local server

```bash
  npm run dev
```
